Pieci.lv

A tiny Mac OS X utility app for listening to the Latvian radio Pieci.lv

View the Project on GitHub hveveris/Pieci.lv

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Installation on Mac OS X only

Drag and drop Pieci.lv.app into your Applications folder and double click it.

Running automatically at startup

  1. Open System Preferences
  2. Click on Users & Groups
  3. Click on Login Items
  4. Use the + button to select and add Pieci.lv.app to the list
  5. Done!
